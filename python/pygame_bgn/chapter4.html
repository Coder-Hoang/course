Chapter 4: Basic Animation

What is our plan for the animation?

First, we need to create a image and blit it on the screen properly.

Next, look at the code down below, it should explain to you the simpliest way and make sure that you have the assets 1, 2, 3, 4.png in you file


import pygame

# Set screen size
screen_width = 800
screen_height = 600
screen = pygame.display.set_mode((screen_width, screen_height))

# Load images (replace 'image1.png', etc. with your actual filenames)
image1 = pygame.image.load('1.png')
image2 = pygame.image.load('2.png')
image3 = pygame.image.load('3.png')
image4 = pygame.image.load('4.png')

# Create a list to hold the images for easy iteration
images = [image1, image2, image3, image4]

# Current image index (starts at 0)
current_image_index = 0

# Animation speed (in milliseconds)
animation_speed = 100  # Adjust for desired speed

# Game loop
running = True
while running:
  for event in pygame.event.get():
    if event.type == pygame.QUIT:
      running = False

  # Clear the screen
  screen.fill((0, 0, 0))  # Black background

  # Draw the current image
  screen.blit(images[current_image_index], (0, 0))

  # Update animation
  pygame.time.delay(animation_speed)  # Wait for some time
  current_image_index = (current_image_index + 1) % len(images)  # Loop through images

  # Update the display
  pygame.display.flip()
